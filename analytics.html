How does this work?

Data Storage Analytics events are stored in this browser's localStorage under the key lul_analytics. This is a client-side demo. For production, connect js/cookies.js to a backend API endpoint (see comments in the file).
Demo Data If no real events exist yet, the dashboard automatically seeds 180 demo events spread over the last 30 days, so you can preview all charts immediately.
Access Control This page is protected by a client-side password gate (default: lul2025). Change the password in analytics.html line ~40. For production, implement server-side authentication.
Production Setup To collect real visitor data, modify js/cookies.js → trackEvent() to POST to your server endpoint instead of using localStorage. All the data structure is already defined.
